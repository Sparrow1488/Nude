Nude.Tg.Bot

Основные функции:
1. Получить мангу в формате статьи в Telegraph (тупа отправляем ссылку)

Под капотом:
1. On /start: создать пользователя
2. При получении ссылки на мангу от пользователя:
   * Запрос на Nude.API для получения манги по Url
   * Если ответ 404: 
      - Делаем запрос на парсинг, создаем RequestTicket с указанием Id пользователя и уникального Id, который отдает Nude.API
      - Пишем пользователю, что данный запрос обрабатывается, просьба подождать
      - Ловим в /callback уникальный Id и Id обработанной манги
      - Делаем запрос на получение манги, вытаскиваем images
      - Создаем статью в Telegraph и пихаем туда ссылки на images (Создаем запись TelegraphManga в БД)
      - Вытаскиваем пользователя по уникальному Id через ParsingRequest и насираем ему в чат ссылкой на статью
   * Если ответ 200:
      - Вытаскиваем по Id из таблицы TelegraphManga по OriginId, берем из нее ссылку и выдаем пользователю
